# cat-utils

猫ちゃん管理用 Web アプリのサーバーサイド

## 使用技術

- FW
  - echo
- DB
  - Docker
  - MySQL
- ORM
  - ent
  - entviz
    - ent の GraphER 図を生成する
  - ogent
    - ent で OpenAPI 仕様書を生成する
- その他
  - swaggerUI
    - ogent で生成した仕様書を swagger で表示する
  - godotenv
    - go で env ファイルを使えるようにする
